## الأمر (Prompt)

أنت مساعد متخصص في تصنيف وتنظيم طلبات المستخدمين لموقع "أئمة التراويح بالرياض" — دليل إلكتروني لمساجد صلاة التراويح وأئمتها في مدينة الرياض.

لدي أكثر من 170 رسالة بريد إلكتروني وردتني من مستخدمين عبر الموقع، لم أعالجها بعد. أحتاج مساعدتك في:

### المطلوب:

**1. التصنيف** — صنّف كل رسالة إلى أحد هذه الأنواع:
- `new_mosque` — طلب إضافة مسجد جديد غير موجود في الدليل
- `imam_change` — إبلاغ عن تغيير إمام في مسجد موجود (سواء إمام جديد أو انتقال إمام معروف)
- `correction` — تصحيح معلومات خاطئة (اسم مسجد، حي، إمام خطأ، الخ)
- `audio_request` — طلب إضافة أو تحديث مقطع صوتي لإمام
- `general` — استفسار عام أو رسالة لا تتعلق بإضافة/تعديل
- `spam` — رسائل غير ذات صلة

**2. استخراج البيانات** — لكل طلب صالح، استخرج البيانات بالتنسيق التالي:

```
رقم: [ترقيم تسلسلي]
النوع: [new_mosque / imam_change / correction / audio_request / general]
المسجد: [اسم المسجد كما ذكره المرسل]
الاسم المعياري للمسجد: [بصيغة: جامع/مسجد/مجمع + الاسم، مع استخدام ة وليس ه]
الحي: [بدون كلمة "حي"، مثال: الملقا وليس حي الملقا]
المنطقة: [شمال / جنوب / شرق / غرب]
الإمام: [بصيغة: الشيخ + الاسم الكامل]
رابط الخريطة: [إن وُجد]
رابط يوتيوب: [إن وُجد]
ملاحظات: [أي تفاصيل إضافية]
```

**3. كشف التكرارات** — حدد:
- رسائل متعددة تطلب نفس المسجد أو نفس الإمام
- طلبات لمساجد **موجودة أصلاً** في الدليل الحالي (انظر القائمة أدناه)
- طلبات تغيير إمام لمسجد **الإمام المذكور هو نفسه الحالي** (لم يتغير فعلياً)

**4. التلخيص النهائي** — قدّم:
- جدول ملخص بجميع الطلبات الصالحة (بدون التكرارات والسبام)
- إحصائيات: عدد كل نوع، أكثر الأحياء طلباً، عدد التكرارات المكتشفة
- قائمة "الإجراءات الفورية" — المساجد الجاهزة للإضافة مباشرة (بيانات كاملة)
- قائمة "تحتاج تحقق" — طلبات ناقصة المعلومات أو متناقضة

### تنبيهات للمطابقة:

- **التطبيع العربي مهم:** تعامل مع `ة`/`ه` و `أ`/`إ`/`ا` و `ال`/`الـ` كأنها متطابقة عند المقارنة
- بعض المرسلين يكتبون اسم المسجد بشكل مختلف (مثلاً "الراجحي بحي الجزيرة" = ID 99)
- بعض المرسلين يذكرون اسم الإمام بدون "الشيخ" — أضفها أنت
- إذا ذكر المرسل حياً غير واضح أي منطقة يتبع، اكتب `تحتاج تحقق` في خانة المنطقة

### شكل المخرج المطلوب:

**أولاً:** جدول شامل لكل الرسائل (رقم، نوع، مسجد، حي، منطقة، إمام، حالة التكرار)

**ثانياً:** قسم التكرارات:
- رسائل مكررة (نفس الطلب من أشخاص مختلفين) — اعرضها مجمّعة
- طلبات لمساجد موجودة أصلاً — أشر إلى الـ ID الموجود

**ثالثاً:** قسم "جاهز للإضافة" — طلبات مساجد جديدة ببيانات كاملة:
```
المسجد: [الاسم المعياري]
الحي: [بدون حي]
المنطقة: [شمال/جنوب/شرق/غرب]
الإمام: [الشيخ ...]
رابط الخريطة: [إن وُجد]
```

**رابعاً:** قسم "تغييرات أئمة" — مرتبة حسب المسجد:
```
المسجد: [الاسم] (ID: [رقم])
الإمام الحالي: [من القائمة أعلاه]
الإمام الجديد المذكور: [ما ذكره المرسل]
عدد البلاغات: [كم شخص أبلغ عن نفس التغيير]
```

**خامساً:** إحصائيات:
- إجمالي الرسائل
- توزيع الأنواع (عدد كل نوع)
- أكثر 5 أحياء طلباً
- عدد التكرارات المكتشفة
- عدد الجاهز للإضافة vs تحتاج تحقق
